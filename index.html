How It Works

Three lines of code and you're live. PayFlow handles the rest.

Install the SDK

Add PayFlow to your project with npm. Works with any JavaScript/TypeScript framework.

npm install @payflow/sdk

Configure processors

Add your existing Stripe, PayPal, and other processor API keys to PayFlow.

const payflow = new PayFlowClient({
stripe: { apiKey: 'sk_...' },
paypal: { apiKey: '...' }
})

Process payments

Use the unified charge API. PayFlow routes to the best processor automatically.

const result = await payflow
.processPayment({
amount: { amount: 49.99, currency: 'USD' }
})

Frequently Asked Questions

¿Cómo manejan los datos de tarjetas de crédito?

Nunca almacenamos números de tarjeta completos. Todo pasa tokenizado a través de procesadores con PCI DSS Level 1. Tu integración existente con Stripe o PayPal sigue siendo exactamente igual de segura.

¿Qué pasa si un procesador falla?

PayFlow detecta la falla en milisegundos y reintenta automáticamente con otro procesador de tu configuración. El cliente nunca ve un error — solo una transacción exitosa.

¿Necesito dejar de usar Stripe/PayPal?

No. PayFlow se conecta a tus cuentas existentes de Stripe, PayPal, Adyen, Square, etc. Sigues usando los mismos procesadores, pero ahora con routing inteligente que optimiza costos.

¿Cuánto cuesta?

Actualmente estamos en fase de desarrollo. El demo es gratuito. El modelo de precio se anunciará en el lanzamiento oficial.

¿Está el código disponible para auditoría?

Sí. El SDK de PayFlow es open source en GitHub. Cualquier desarrollador puede revisar el código, verificar la seguridad, y contribuir.
